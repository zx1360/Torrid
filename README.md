# Torrid介绍

## 功能

- 计划是一个集成记录打卡信息并总览、记录随笔并打标签等等功能的主要适配手机平台的多段应用
- 还有其他若干很多想做的内容塞进去......

## 使用技术

### UI框架

- **Flutter** 高性能跨平台UI框架, 基于dart语言, 包揽了传统网页框架的html/css/js三者.
  - Hive 本地存储工具.
  - riverpod 状态管理

### 后端(待做)

- **nodejs** 作为服务端.
  - 计划是运行在PC端, 处理应用的请求来进行数据备份之类的.
