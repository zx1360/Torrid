# 各页面简介

## 积微:

**包含两大板块, 打卡和试炼**

### 打卡

- 可以设置1-5个任务(标题, 描述, 图片), 每天进行勾选打卡
- 实时显示已打卡天数/全达成天数/最长连续打卡天数
- 总览页可以查看以往样式的记录, 并回看以往的所有记录.
